# remove-contaminant-sequences
1. The python script '06.Programs_for_remove_contaminant_sequences.py' is used for remove contaminant sequences.
Running:
python 06.Programs_for_remove_contaminant_sequences.py <target_hits.m6.txt> <other_hits.m6.txt> <gene.gff> <genome.fa>

2. Example:
python 06.Programs_for_remove_contaminant_sequences.py invertebrates_hits.m6.txt non_invertebrates_hits.m6.txt DdB_gene.gff DdB_genome.fa
